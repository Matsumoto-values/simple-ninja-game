# SHINOBI RUSH 〜疾風の刃〜

ブラウザで動く 3D TPS 忍者アクションゲーム。
敵の弾幕をスライドダッシュ・二段ジャンプ・壁走りでかいくぐり、立ちはだかる敵をスタイリッシュに斬り捨てて、体力が尽きる前に彼方の大鳥居(ゴール)を目指せ。

## 遊び方(開発)

Vite ベースのプロジェクトです。

```bash
npm install
npm run dev      # 開発サーバー(http://localhost:5173)
npm run build    # 本番ビルド → dist/
npm run preview  # ビルド結果の確認
```

Vercel には `vercel.json` の設定でそのままデプロイできます(フレームワーク: Vite、`npm run build` → `dist/`)。

## 操作方法

| 操作 | キー |
| --- | --- |
| 移動 | WASD / 矢印キー |
| ジャンプ(空中でもう一度で二段跳び) | Space |
| スライドダッシュ(無敵回避・空中可) | Shift |
| 壁走り(ウォールラン) | 空中で壁際に張り付いて前進 |
| 斬撃(近くの敵へ疾風斬り) | クリック / J / Enter |
| やり直し | R |

### スマートフォン(タッチ操作)

タッチ端末では画面にバーチャルパッドが自動表示されます(タイトル画面の「操作」または画面右下の 🎮 ボタンでON/OFF切替可能)。

- **左のスティック**: 移動(アナログ入力)
- **斬**(赤・大): 斬撃
- **跳**(青): ジャンプ(空中でもう一度で二段跳び)
- **疾**(水色): スライドダッシュ
- 画面タップでも斬撃できます

## 難易度

| 難易度 | 内容 |
| --- | --- |
| 初伝 | 敵はまばら。単発の火球と手裏剣が中心。被ダメージも軽減 |
| 中伝 | 3方向の扇撃ち・巨大弾・地走りの衝撃波が加わる |
| 皆伝 | **弾幕地獄**。5方向扇撃ち・3連射・道幅いっぱいの弾のカーテン(面制圧)が飛び交う。二段ジャンプ・壁走り・ダッシュ無敵を使いこなさないと抜けられない |

## 弾の種類と避け方

- **火球**(橙の小弾): 狙い撃ち。横移動やダッシュで避ける
- **手裏剣**(高速回転): 速い。早めの見切りを
- **妖力弾**(巨大な紫弾): 遅いがデカい。当たると痛い
- **衝撃波**(地を走る横長の波): **ジャンプで飛び越える**
- **弾のカーテン**(道幅いっぱいの赤玉の列): **ジャンプで越えるかダッシュ無敵ですり抜ける**

## ゲームのルール

- 体力(HP)は 100。スライドダッシュ中は無敵。弾がかすめると「見切り」となり、ダッシュのクールタイムが短縮される
- 斬撃は前方の敵を自動ロックオンし、一瞬で踏み込んで一刀両断。ヒットストップとスローモーションの爽快演出つき
- 道中に浮かぶ緑の霊珠で体力回復(+25)
- 体力が尽きる前にゴールの大鳥居へたどり着けばクリア。タイム・撃破数・残り体力が記録される

## タイトル画面のキャラクター画像の差し替え

タイトル画面右側に表示される画像は `assets/title-character.png` です(現在はダミー画像)。
同じパスに任意のPNGを上書きすれば差し替わります。

- **透過PNG**(背景なし)推奨
- 縦長(目安 630×930 前後、縦横比はずれていてもOK。`object-fit: contain` で下端揃えに収まります)

## 技術構成

- [Three.js](https://threejs.org/) r180 + [Vite](https://vitejs.dev/)(npm 管理・ビルドあり)
- **セルルック描画**: MeshToonMaterial(4段グラデーション)+ 背面法の輪郭線
- **ポストプロセス**: [postprocessing](https://github.com/pmndrs/postprocessing) によるブルーム + ビネット、ACES トーンマッピング
- **リアルタイムシャドウ**(PCFソフト、プレイヤー追従のシャドウカメラ)
- **プロシージャルテクスチャ**(石畳・漆喰・地面ノイズを canvas で生成、外部画像アセット不要)
- **GPUインスタンシング**(竹林・桜/紅葉並木・塀の柱を一括描画)
- グラデーションの空ドーム(カメラ追従シェーダー)
- 効果音は WebAudio API でリアルタイム合成
- モバイルでは影の解像度・MSAA・ピクセル比を自動で軽量化
