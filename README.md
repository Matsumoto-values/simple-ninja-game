# SHINOBI RUSH 〜疾風の刃〜

ブラウザで動く 3D TPS 忍者アクションゲーム。
敵の弾丸をスライドダッシュでかいくぐり、立ちはだかる敵をスタイリッシュに斬り捨てて、体力が尽きる前に彼方の大鳥居(ゴール)を目指せ。

## 遊び方

ES Modules を使っているため、ローカルの HTTP サーバー経由で開いてください。

```bash
# リポジトリ直下で
python3 -m http.server 8000
# または
npx serve .
```

ブラウザで http://localhost:8000 を開くとゲームが始まります。

## 操作方法

| 操作 | キー |
| --- | --- |
| 移動 | WASD / 矢印キー |
| スライドダッシュ(無敵回避) | Shift / Space |
| 斬撃(近くの敵へ疾風斬り) | クリック / J / Enter |
| やり直し | R |

## ゲームのルール

- 体力(HP)は 100。敵の弾丸(-12)や鬼武者の体当たり(-25)でダメージを受ける。
- **スライドダッシュ**中は無敵。弾がかすめると「見切り!」となり、ダッシュのクールタイムが短縮される。
- **斬撃**は前方の敵を自動でロックオンし、一瞬で踏み込んで一刀両断する。ヒットストップとスローモーションで爽快感を演出。
- 道中に浮かぶ緑の霊珠を取ると体力が 25 回復する。
- 体力が尽きる前にゴールの大鳥居へたどり着けばクリア。タイム・撃破数・残り体力が記録される。

## 技術構成

- [Three.js](https://threejs.org/) r160(`lib/` に同梱、ビルド不要)
- 効果音は WebAudio API でリアルタイム合成(外部アセットなし)
- 依存パッケージ・ビルド工程なしの静的サイト。GitHub Pages にそのままデプロイ可能。
